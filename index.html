Rhizome Project

Rhizome is Mobile Health using AngularJS and HL7 FHIR (Fast Healthcare Interoperability Resources). It stands for Resource-based Health Information Zone for the Mobile Enterprise.

  • Presentation Slides: The Next Five Years
  • Rhizome Questions4Care Demonstration
  • Rhizome Timelines4Care Demonstration

For more information, refer to the demonstrations above. Also refer to the embedded demos below, which showcase both Q4C and T4C, and my Mobile First Interoperability poster presentation, both of which are also in this project.

Rhizome is a library of templates and services I am building using AngularJS and other JavaScript libraries, guided by the notion of creating a set of client-side components to support standards-based mobile and social application development for the Healthcare industry.

Questions for Care

Questions for Care (Q4C) is a healthcare demonstrator combining an industry standard (HL7 FHIR), a JavaScript framework (AngularJS), and Scalable Vector Graphic animation (SnapSVG). See the Rhizome repository.
